# 🌆 Street Sweep Pro

A browser-based street cleaning game controlled using AI hand gestures.

## 🎮 Features

- 🖐️ Hand gesture controls
- 🎮 Manual button controls
- 🌆 Animated city environment
- ✨ Particle effects
- 🔊 Sound effects
- 🏆 Score tracking
- 📱 Responsive design
- ♻️ Environmental awareness gameplay

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas
- MediaPipe Hands
- Web Audio API

---

## ✋ Gesture Controls

| Gesture | Action |
|----------|--------|
| ✋ Open Palm | Walk Forward |
| ✊ Closed Fist | Stop |
| ✌️ Two Fingers | Pick up Trash |
| ☝️ One Finger | Drop Trash into Bin |

---

## 🎮 How to Play

1. Choose Camera Mode or Manual Mode.
2. Walk through the city.
3. Pick up litter.
4. Move to the recycling bin.
5. Drop the trash.
6. Reach the finish line with the highest score
